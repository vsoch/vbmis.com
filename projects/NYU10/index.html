NYU10 DETAILED REPORT

This report shows significant structural deficits and differences in functional networks for a 10 subject subset of the NYU ADHD200 dataset.  The goal is to look for overlaps between structural and functional deficits.
STRUCTURAL DEFICITS Significant structural differences in gray and white matter as reported in the literature
FUNCTIONAL NETWORKS Significant functional network differences between ADHD and Control derived with multi-session temporal concatenation group ICA analysis and dual regression
FUNCTIONAL COMPONENT OVERVIEW: Includes a summary of dual regression results and anatomical areas, and the most likely overlaps with meta analysis results
LOCAL MAX VOXELS ORDERED BY F VALUE: Includes max voxel results ordered by F value, both before and after visual filtering

FUNCTIONAL NETWORKS

For each component (a significant functional network found to exist for the combined group of ADHD and Control) a dual regression was run to find significant differences between the two groups.  Therefore, each set of images below represents the significant differences in functional networks between individuals with ADHD and Control.  Since the goal of this report is to compare these functional differences to structural deficits, the image detailed above from the meta analysis is also included (RED).

Each image result is followed by a table, which shows the top six local maxima reported for the dual regression result.  It should be noted that these dual regression images are all p corrected and thresholded.
